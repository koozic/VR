# 작업 이력

이 문서는 완료되었거나 중단된 작업의 이력을 기록한다. 다음 작업자가 같은 조사를 반복하지 않도록 확인한 사실, 실패한 시도, 후속 작업을 함께 남긴다.

## 작성 규칙

- [ ] 작업 종료 시 반드시 새 로그를 추가한다.
- [ ] 수정, 생성, 삭제 파일을 분리해 적는다.
- [ ] 실패한 시도가 없으면 "없음"으로 적는다.
- [ ] 발견한 문제가 없으면 "없음"으로 적는다.
- [ ] 추측은 기록하지 않는다.
