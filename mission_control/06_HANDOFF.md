작업 종료 시마다 최신 상태로 덮어쓴다.

| 항목 | 내용 |

## 바로 알아야 할 것

| 주제 | 내용 |
| --- | --- |
| 작업 범위 | 작품 편집 관리자 UI, 백엔드 health 상태 조회 연동 |
| 변경 의도 | H2 임시 DB와 Oracle 공용 DB를 헷갈려 잘못 수정하는 일을 줄임 |
| 건드리지 않은 것 | 백엔드 작품 저장 로직, DB 설정, 실행 스크립트 |
| 불확실한 것 | 현재 자리에서 Oracle DB 접속이 가능한지, 실제 팀 공용 DB에 저장되는지 수동 확인 필요 |
