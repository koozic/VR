# 인수인계 패킷

이 문서는 다음 작업자가 지금 상태를 빠르게 이어받기 위한 짧고 정확한 인수인계 문서다.  
작업 종료 시마다 최신 상태로 덮어쓴다.

## 현재 인수인계

| 항목 | 내용 |
| --- | --- |
| 마지막 갱신일 | 2026-07-08 |
| 마지막 작업자 | Codex |
| 현재 초점 | 레트로게임관 포스터와 가까운 게임 설명 연결 |
| 현재 상태 | 구현 및 프론트 빌드 검증 완료, 실제 장면 수동 확인 필요 |
| 다음 추천 작업 | 각 포스터에 접근해 가장 가까운 게임기의 제목·설명이 표시되는지 확인 |

## 바로 알아야 할 것

| 주제 | 내용 |
| --- | --- |
| 작업 범위 | `createRetroDecor.js`, `retroGalleryRuntime.js` |
| 변경 의도 | 18×22 레트로게임관의 외곽·후면·바닥 여백을 장식으로 채움 |
| 건드리지 않은 것 | 게임 URL, 게임기 상호작용, 공유 seed, 다른 전시관 런타임 |
| 불확실한 것 | 실제 플레이 화면에서의 네온 밝기와 오브젝트 시야 가림 정도 |

## 다음 작업자가 할 일

- [ ] 레트로게임관으로 이동해 바닥 네온과 장식이 정상 표시되는지 확인한다.
- [ ] 후면 포탈이 ARCADE 간판과 스피커에 가려지지 않는지 확인한다.
- [ ] 중앙 게임기 5대 접근과 이동 동선이 유지되는지 확인한다.
- [ ] 과하게 밝거나 깜빡이는 재질이 있으면 `createRetroDecor.js`의 opacity를 조정한다.
