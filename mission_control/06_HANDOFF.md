이 문서는 다음 작업자가 지금 상태를 빠르게 이어받기 위한 짧고 정확한 인수인계 문서다.  
작업 종료 시마다 최신 상태로 덮어쓴다.

## 현재 인수인계

| 항목 | 내용 |
| --- | --- |
| 마지막 갱신일 | 2026-07-06 |
| 마지막 작업자 | Codex |
| 현재 초점 | `mission_control` 중앙 통제실 문서 체계 생성 |
| 현재 상태 | 문서 템플릿 생성 완료 |
| 다음 추천 작업 | 프로젝트 고유 목표, 금지 원칙, 실제 실행/검증 명령 채우기 |

## 바로 알아야 할 것

| 주제 | 내용 |
| --- | --- |
| 작업 범위 | `mission_control` 폴더 내부 문서 체계 |
| 변경 의도 | AI 작업자 교체 시 컨텍스트 손실을 줄이고, 임의 수정과 이력 누락을 방지 |
| 건드리지 않은 것 | 애플리케이션 코드, 빌드 설정, 실행 스크립트 |
| 불확실한 것 | 팀의 실제 운영 우선순위, 배포/시연 조건, 금지 영역 |
