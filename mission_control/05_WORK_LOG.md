# 작업 이력 로그

이 문서는 누가 무엇을 왜 바꿨는지 추적하는 작업 로그다.  
작업이 끝날 때마다 최신 항목을 위에 추가한다.

## 로그 작성 규칙

| 규칙 | 설명 |
| --- | --- |
| 최신순 | 새 로그는 가장 위에 추가한다. |
| 사실 중심 | 실제로 한 일과 확인한 일을 분리해서 쓴다. |
| 검증 명시 | 테스트/빌드/수동 확인을 명확히 적는다. |
| 미검증 기록 | 못 해본 것은 못 했다고 적는다. |
| 다음 작업 연결 | 다음 사람이 바로 시작할 수 있는 행동을 적는다. |
